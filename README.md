# 감정 기반 맛집 추천 시스템

> FastAPI에서 GET/POST API를 구현하고, 사용자 입력을 AI 추론 결과로 연결했으며, CBF 추천 로직과 카카오 지도 API 연동까지 담당한 감정 기반 추천 서비스

## 프로젝트 소개

이 프로젝트는 사용자가 입력한 감정 문장을 바탕으로 감정을 분석하고, 그 결과에 맞는 음식 및 카테고리를 추천한 뒤 최종적으로 관련 맛집 정보를 제공하는 서비스입니다.

백엔드는 FastAPI를 기반으로 구성했으며, 사용자가 입력한 데이터를 AI 모델로 전달하고 결과를 JSON 형태로 반환해 웹 화면에 출력되도록 구현했습니다.  
또한 감정 분석 결과를 기반으로 음식과 카테고리를 추천하는 CBF(Content-Based Filtering) 로직을 구현했고, 추천된 맛집 정보는 카카오 지도 API와 연동하여 위치를 시각적으로 확인할 수 있도록 구성했습니다.

---

## 내가 한 일

- FastAPI 기반으로 결과 조회용 GET API와 사용자 입력 처리용 POST API를 구현했습니다.
- 사용자가 감정 문장을 입력하면 POST 요청으로 데이터를 받아 AI 모델 추론이 수행되고, 결과가 JSON 형태로 반환되어 화면에 출력되도록 연결했습니다.
- GET 요청을 통해 감정 분석 및 추천 결과를 확인할 수 있도록 응답 구조를 구성했습니다.
- KoBERT 감정 분석 결과를 기반으로 음식과 카테고리를 추천하는 CBF(Content-Based Filtering) 로직을 구현했습니다.
- 추천 데이터를 프론트엔드에서 바로 사용할 수 있도록 응답 포맷을 정리했습니다.
- 카카오 지도 API를 연동해 추천 맛집의 위치를 지도에서 시각적으로 확인할 수 있도록 구현했습니다.

---

## 핵심 기능

### 1. 감정 분석 결과 출력
사용자가 텍스트 입력창에 감정 문장을 입력하면 FastAPI 서버가 POST 요청으로 데이터를 받아 AI 모델 추론 결과를 반환합니다.  
반환된 결과는 JSON 형태로 웹 화면에 출력되며 사용자는 자신의 감정 분석 결과를 바로 확인할 수 있습니다.

### 2. 결과 조회용 API 구성
분석 및 추천 결과를 GET 요청으로 확인할 수 있도록 API를 구성하여 프론트엔드에서 필요한 데이터를 안정적으로 조회할 수 있게 했습니다.

### 3. CBF 기반 음식/카테고리 추천
KoBERT 감정 분석 결과를 입력값으로 받아 사용자 감정에 맞는 음식과 카테고리를 추천하는 CBF 로직을 구현했습니다.  
추천 결과는 JSON 응답으로 반환되며 화면에서 사용자가 다음 선택을 할 수 있도록 연결됩니다.

### 4. 카카오 지도 API 연동
추천된 맛집 데이터는 카카오 지도 API와 연동하여 지도 위에 위치를 시각적으로 표시했습니다.  
사용자는 추천 결과를 텍스트뿐 아니라 지도 기반 UI로도 확인할 수 있습니다.
